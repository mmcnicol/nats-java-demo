# nats-java-demo


## versions

wildfly 27  
java ee 10  
nats 2.16.5


## start wildfly & NATS

docker compose up


## to send a message, open this url in a web browser

http://localhost:8080/nats-java-demo-1.0-SNAPSHOT/api/v1/queue/send2


## wildfly console

http://localhost:9990/console


## nats dashboard

http://localhost:8222/
